jar -tf xxx.jar 可以查看 jar 文件内容

jar -xf xxx.jar 可以解压 jar 文件

java 文件在不同的包中

上面是理想的情况，所有类在一个文件夹下（包）下，但是这是很少见的，为了代码规范及防止包名冲突等原因，一般情况下我们都会分包，根据业务或代码架构的层级会把 java 文件放到不同的包里。因此生成的 class 文件也在不同的包里。
